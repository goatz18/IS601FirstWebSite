Git Command Line and First Website Assignment

Purpose

The purpose of this web page is to gain experience in learning basic html and basic Linux, Vi, and Git commands, all at the same time. Great assignment ;)

Linux Commands

pwd – where you are in the file system
cd – Change Directory
clear – clears up the terminal screen
mkdir NAME <replace with desired name – Make Directory
ls – list files
git init – initialize a repository
vi FILENAME <replace with the actual file name – enter vi edit for said file

Vi Commands

i – insert mode
escape – get out of insert mode
shift key + colon – command prompt for vi
w – command to write
q – command to quit
wq – Write and Quit

Git Commands

git init – Create the repository
git add filename – add the file(s) to git in preparation for a commit
git commit -m “MESSAGE” – creates a commit, to save changes, with a desired commit message
git checkout -b BRANCHNAME <replace with desired branch name – make a new branch and checkout
git checkout BRANCHNAME – checkout branch
git merge BRANCHNAME – merge the branch
Note: You must currently be on the branch that you want to merge.
